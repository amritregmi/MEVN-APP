## Installation

    ```
    Clone The Repo
       $ git clone https://github.com/amritregmi/MEVN-APP.git

    Get into Client Part
       $ cd client-vue

    Run vue Client 
       $ npm run serve
    
    Run json server
       $ npm run backend 
    
    OR 

    Run node server 
        $ cd server-nodejs
        $ npm run start:prod
    
    (Use POSTMAN or INSOMANIA TO TEST NODE REST API)

    ```
