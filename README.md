## Installation

    ```
    Clone The Repo
       $ git clone https://github.com/amritregmi/MEVN-APP.git

    Get into Client Part
       $ cd client-vue
       $ npm install

    Run vue Client (VUE JS RUNS)
       $ npm run serve
    
    Run node server (NODE SERVER - MongoDB runs)
        $ cd server-nodejs
        $ npm install
        $ npm run start:dev
    
    ()

    ```
